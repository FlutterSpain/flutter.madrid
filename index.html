Tu comunidad de #Flutter en Madrid

Flutter es un SDK open source creado principalmente por Google para construir increíbles aplicaciones compiladas nativamente para Android, iOS, web y escritorio con el mismo código fuente.

Somos una comunidad local creada por y para desarrolladores interesados en el desarrollo de aplicaciones con Flutter y Dart.

¿Qué hacemos?

Meetups

Charlas mensuales con temas de interés para la comunidad y la última actualidad.

Soporte

Cuéntanos tu duda o problema y buscamos una solución juntos.

Talleres

Potenciamos el uso de Flutter realizando talleres presenciales.

Formación

Colaboramos con empresas y escuelas para enseñar a más personas.
